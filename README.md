# Конформные отображения | Лабораторная работа №2

Данный проект реализует визуализацию конформных отображений для комплексного анализа. Он позволяет демонстрировать, как функции отображают области на комплексной плоскости.

## Возможности

- Интерактивная визуализация конформных отображений
- Демонстрация динамического перехода одной комплексной функции в другую
- Адаптивный интерфейс для исследования свойств отображений (возможности остановить анимацию, поменять скорость воспроизведения для лучшего рассмотрения процесса)

## Технологии

- HTML, CSS, JavaScript
- Библиотеки для визуализации — Chart.js
