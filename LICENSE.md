# Лицензия

Проект создан в рамках курса по комплексному анализу студентами Университета ИТМО.

Реализация лабораторной работы распространяется по лицензии **CC BY-NC 4.0**.

Использование материалов проекта разрешено только в некоммерческих целях при обязательном указании авторов и ссылки на оригинальный репозиторий: [@TheGeniusOfEternity](https://github.com/TheGeniusOfEternity), [@scobca](https://github.com/scobca)

Разрешается модификация и улучшение проекта при сохранении указания исходных авторов и явном указании произведённых изменений.
